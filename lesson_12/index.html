CSS Transform ( HOVER )

Translate (x, y)

translate

Translate- Зміщує елемент на нове місце, переміщаючи відносно початкового положення об'єкта вправо та вниз, використовуючи координати X та Y. Записується так translate (x,y).

Scale (x, y)

Scale

Scale- Масштабує елементи, роблячи їх більшими або меншими.

Rotate (n + deg)

Rotate

Rotate- Повертає елементи.

Skew (x + deg, y + deg)

Skew

Skew- Деформує (нахиляє) елементи.

Matrix (a, c, b, d, x, y)

Matrix

Matrix- Дозволяє об'єднати кілька функцій 2D-трансформацій в одній.

Transform-origin

Transform-origin

Transform-origin- Дозволяє змістити центр трансформації, щодо якого відбувається зміна положення/розміру/форми елемента.

Perspective, perspective-origin

Perspectiven

Perspective- Встановлення глибини та зміна точки початку координат. !! Задаються батьківському елементу!

Transform-style

Transform-style

Transform-style- Визначає, як дочірні елементи мають відображатися у 3D-просторі. ! ! Задається батьківському елементу разом з perspective !

Backface-visibility

Backface-visibility

Backface-visibility- Відображення зворотного боку об'єкта.
